# 💰 VeriProp Nigeria — Fee Structure Recommendation

## Based on: Nigerian market data, global best practices, marketplace economics

---

## TL;DR — My Recommendation

| Decision | Recommendation | Why |
|---|---|---|
| **Who pays platform fee?** | **BUYER pays** | This is Nigerian standard. Buyers/tenants already expect to pay agent fees on top of property price. Adding a small platform fee is natural. Landlords receive 100% — attracts more listings. |
| **Platform fee %** | **2.5%** | Sweet spot: low enough not to scare buyers, high enough to sustain the business. Lower than any agent's fee, so buyers see it as good value for escrow protection. |
| **Agent commission** | **5-10% (agent sets)** | Nigerian standard is 10% for rentals, 5% for sales. Let agents set their own rate within this range. |
| **Direct listing fee** | **2.5% only** | When landlord lists without an agent, buyer pays only the 2.5% platform fee. No agent commission = cheaper for buyer, incentivizes direct listings. |

## The Full Analysis

### What Nigerian Buyers Already Pay (market reality)

From PropertyPro, BuyLetLive, and NIESV data:

**Rental Transaction:**
```
Annual rent:           ₦2,000,000
Agent fee (10%):       ₦200,000  ← Paid by tenant (Nigerian standard)
Legal/agreement fee:   ₦50,000-₦100,000
Caution deposit:       ₦200,000
TOTAL tenant pays:     ₦2,450,000 (122.5% of rent)
```

**Sale Transaction:**
```
Property price:        ₦50,000,000
Agent fee (5%):        ₦2,500,000  ← Paid by buyer (Nigerian standard)
Legal fees (7.5%):     ₦3,750,000
Stamp duty (2%):       ₦1,000,000
Registration (3%):     ₦1,500,000
TOTAL buyer pays:      ₦58,750,000 (117.5% of price)
```

**Key insight:** Nigerian buyers already pay 15-25% ON TOP of the property price. Adding a 2.5% platform fee is small in comparison, and they get escrow protection in return.

**Key insight:** VeriProp at 2.5% would be the CHEAPEST platform fee in any marketplace category. This is intentional — it's a trust play, not a fee play.

### Why BUYER Should Pay (Not Seller/Agent)

| Reason | Explanation |
|---|---|
| **Nigerian standard** | Tenants/buyers already pay all extra fees. This is culturally expected. |
| **Landlord attraction** | If landlords get 100% of their asking price, they'll list more properties. More listings = more buyers. |
| **Agent attraction** | If agents keep their full commission, they'll bring properties to VeriProp. |
| **Value exchange** | Buyer is getting: escrow protection, verified seller, dispute resolution, legal proof of payment. Worth 2.5%. |
| **Simplicity** | One fee, one payer, no confusion. |

### Why 2.5% Is The Right Number

| % | Monthly revenue (10 transactions × ₦10M avg) | User perception |
|---|---|---|
| 1% | ₦1,000,000 | Too low — hard to sustain business |
| **2.5%** | **₦2,500,000** | **Sweet spot — affordable for buyer, sustainable for business** |
| 3% | ₦3,000,000 | Still reasonable |
| 5% | ₦5,000,000 | Starts to feel expensive on big transactions |
| 10% | ₦10,000,000 | Same as agent fee — users will revolt |

**2.5% on a ₦2M annual rent = ₦50,000 platform fee.** That's LESS than most agreement fees tenants already pay. And they get escrow protection worth far more.

**2.5% on a ₦50M property sale = ₦1,250,000 platform fee.** Compared to ₦2.5M-₦5M agent fee they're already paying, this is small.

## Recommended Fee Breakdown

### Scenario 1: With Agent (most common)
```
Property price:         ₦10,000,000
Buyer pays:
  → Property price:     ₦10,000,000 (goes to Landlord via escrow)
  → Agent commission:   ₦500,000 - ₦1,000,000 (5-10%, goes to Agent)
  → Platform fee:       ₦250,000 (2.5%, goes to VeriProp)
  → Paystack fee:       ~₦152,000 (1.5% + ₦100, auto-deducted)
  
TOTAL buyer pays:       ₦10,750,000 - ₦11,250,000
Landlord receives:      ₦10,000,000 (100%)
Agent receives:         ₦500,000 - ₦1,000,000
VeriProp receives:      ₦250,000
```

### Scenario 2: Direct Listing (no agent)
```
Property price:         ₦10,000,000
Buyer pays:
  → Property price:     ₦10,000,000 (goes to Landlord via escrow)
  → Platform fee:       ₦250,000 (2.5%, goes to VeriProp)
  → Paystack fee:       ~₦152,000 (1.5% + ₦100, auto-deducted)
  
TOTAL buyer pays:       ₦10,250,000
Landlord receives:      ₦10,000,000 (100%)
VeriProp receives:      ₦250,000
```

### Scenario 3: Rental
```
Annual rent:            ₦2,000,000
Tenant pays:
  → Rent:               ₦2,000,000 (goes to Landlord via escrow)
  → Agent commission:   ₦200,000 (10%, goes to Agent)
  → Platform fee:       ₦50,000 (2.5%, goes to VeriProp)
  → Paystack fee:       ~₦30,100 (auto-deducted)
  
TOTAL tenant pays:      ₦2,250,000
Landlord receives:      ₦2,000,000 (100%)
Agent receives:         ₦200,000
VeriProp receives:      ₦50,000
```

## VAT and Tax Considerations

| Tax | Rate | Who Pays | How |
|---|---|---|---|
| VAT | 7.5% | Applied on platform fee only | VeriProp collects and remits to FIRS |
| WHT | 5-10% | On agent commission | Agent's responsibility |

**VeriProp's VAT obligation:** 7.5% of ₦250,000 = ₦18,750 remitted to FIRS.
**VeriProp keeps:** ₦250,000 - ₦18,750 = ₦231,250 net per ₦10M transaction.

## Final Recommendation: IMPLEMENT THIS

```
PLATFORM FEE:     2.5% paid by buyer/tenant
AGENT COMMISSION: 5-10% paid by buyer/tenant (agent sets rate)
DIRECT LISTING:   2.5% only (no agent fee)
LANDLORD PAYOUT:  100% of asking price (always)
VAT:              7.5% on platform fee (VeriProp remits)
PAYSTACK FEE:     1.5% + ₦100 (auto-deducted by Paystack)
```

This model:
✅ Follows Nigerian market norms (buyer pays extras)
✅ Attracts landlords (they get full amount)
✅ Attracts agents (they keep full commission)
✅ Lowest platform fee of any marketplace (2.5%)
✅ Buyer gets real value (escrow + verification + dispute resolution)
✅ Sustainable revenue for VeriProp
✅ Simple to understand and implement
